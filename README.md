# Rice Cooker

Este programa realiza a simulação de uma panela elétrica de arroz em um microprocessador `PIC18F4520`. Desenvolvido durante as aulas de Programação Embarcada e Laboratório na Unifei - Universidade Federal de Itajubá

## Estrutura
Arquivos `README` e `Relatório` na pasta raíz, bibliotecas na pasta `Header files` e desenvolvimento das bibliotecas e `main` na pasta `Source files`.
